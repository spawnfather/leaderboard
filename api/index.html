SpawnBoard API Documentation

This page documents the available API endpoints for retrieving leaderboard data from SpawnBoard. All responses are in JSON format.

Endpoint: /leaderboard/all

Returns the entirety of the leaderboard as JSON. This endpoint provides a complete list of all servers in the leaderboard.

Examples

Example 1: Basic Fetch

import requests

# Replace with your API base URL
url = "https://api.spawnboard.com/leaderboard/all"

response = requests.get(url)
if response.status_code == 200:
    data = response.json()
    print(data)  # Output the full leaderboard JSON
else:
    print("Error:", response.status_code)

import requests
from requests.exceptions import RequestException

try:
    url = "https://api.spawnboard.com/leaderboard/all"
    response = requests.get(url, timeout=10)
    response.raise_for_status()
    data = response.json()
    print("Leaderboard:", data)
except RequestException as e:
    print("Failed to fetch leaderboard:", str(e))

Endpoint: /leaderboard/top/{amount}

Returns the top {amount} servers from the leaderboard as JSON. Replace {amount} with the number of top servers you want to retrieve (e.g., /leaderboard/top/10).

Examples

Example 1: Fetch Top 5

import requests

amount = 5
url = f"https://api.spawnboard.com/leaderboard/top/{amount}"

response = requests.get(url)
if response.ok:
    top_servers = response.json()
    print(top_servers)
else:
    print("Error fetching top servers")

Example 2: Dynamic Amount with Validation

import requests

def get_top_servers(amount):
    if not isinstance(amount, int) or amount < 1:
        raise ValueError("Amount must be a positive integer")
    url = f"https://api.spawnboard.com/leaderboard/top/{amount}"
    response = requests.get(url)
    return response.json() if response.status_code == 200 else None

# Usage
print(get_top_servers(10))
