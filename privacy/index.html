Privacy Policy

Last updated: October 28, 2025

Spawn Board does not collect, store, or process any personal data. This Privacy Policy explains exactly what information we handle and why — all of it is public and non-personal.

1. What We Do

Spawn Board is a public leaderboard that displays Discord servers based on:

  • Public invite codes listed on Disboard.org
  • Approximate member counts retrieved via Discord's public API

We automatically fetch and display this data. No login, account, or personal information is required to view the site.

2. The Submit Server Form

The form at /submit allows anyone to suggest a Discord server for inclusion. It collects only:

  • Server Name (public)
  • Public Invite Link (must be permanent and public)
  • Short Description (optional, public)

No email address, IP address, or personal identifier is collected or stored. Submissions are reviewed manually and only approved if the invite is valid and public.

3. Data We Display (All Public)

The leaderboard shows only:

  • Server name
  • Public invite link
  • Approximate member count (from Discord API)
  • Short description (if provided)

This information is already publicly available via Disboard and Discord. We do not access private channels, user lists, or messages.

4. Cookies & Tracking

We use no cookies, analytics, or tracking scripts.

The only data stored locally on your device is your dark/light theme preference using localStorage. This is optional and never sent to our server.

5. Data Storage & Retention

  • Approved servers remain on the leaderboard until the invite expires or is removed by the site operator.
  • Rejected or invalid submissions are discarded immediately.
  • No logs of form submissions or visitor activity are kept.

6. Third-Party Services

We interact with:

  • Disboard.org – to discover public server listings
  • Discord API – to get approximate member counts using public invite links

We do not send them any user data. They only return publicly available server info.

7. Children's Privacy

Since no personal data is collected, our site is suitable for all ages. We comply with COPPA by not gathering any information from children under 13.

8. Changes to This Policy

We may update this policy if our service changes. The latest version will always be available at /privacy.

9. Removal Requests

If you want your server removed from the leaderboard, submit a new form at /submit with "Remove" in the description and your server invite. We’ll remove it promptly.
